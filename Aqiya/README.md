# Aqiya

Landing page and brand asset draft for **Aqiya**, an AI sourcing agent for vacant-home deals.

## Files

- `index.html` - Static landing page.
- `styles.css` - Minimal editorial infographic styling.
- `logo.svg` - Primary logo concept.

## Design Notes

- The `q` mark combines a map pin, magnifier, and door-handle detail.
- Palette follows a white-paper style: white, neutral black, soft gray, and orange-red `#f9604d`.
- Layout uses large left-aligned headings, numbered sections, flat dashboard diagrams, and thin line icons.
